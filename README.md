# Godot写的扫雷小游戏~

与Godot-Snake采取一致的政策~

本开源代码的核心逻辑部分仅使用短短的400多行（不计注释与空行）解决。存放在GameScene.gd类。

尽情观看吧~

二进制下载：[蓝奏云](https://wwdy.lanzoub.com/ikZiE2lbhoub)，密码：6amm
